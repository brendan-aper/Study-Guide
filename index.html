/HTML

  • Hypertext Markup Language (HTML) is used to create the structure of a webpage.
  • It is made up of elements(Text, Image, Header, List,etc) which work together to provide content and a basic framework.
  • HTML is used to build a frame for a webpage
  • The head element contains information about the webpage.
  • The body element represents the visible content to the user.
  • Section elements are semantic elements that define a section in a document that contains thematic content.

Git

  • git is a version control system that lets you manage and keep track of each version of your project.
  • git commands ALWAYS start with git.
  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git add -A: adds ALL(-A) changes in the current working branch to the staging area.
  • git commit -m: commits all changes added with a message(-m) that describes what the commit contains.
